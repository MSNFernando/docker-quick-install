# Docker quick install script

### Get started
```
# Clone repo
git clone https://github.com/MSNFernando/docker-quick-install.git

# Change dir
cd docker-quick-install/

# Make executable
sudo chmod +x install.sh

# Install docker
./install.sh
```

Clean up by deleting the install.sh file.
